# LoL Esports — Live Tournament Tracker

A single-file, auto-updating bracket & schedule tracker for League of Legends esports
(2026 season). Region tabs, season stages (First Stand · Road to MSI · MSI · Summer · Worlds),
live match status with stream links, interactive brackets with MSI seed boxes, and
regular-season standings.

**Live site:** see the GitHub Pages link in the repo's About section.

Data comes from the unofficial LoL Esports feed (esports-api.lolesports.com) and is fetched
client-side — no backend. Not affiliated with or endorsed by Riot Games.
